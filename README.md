# distributed-lock

#### 分布式锁实现分析

| 方式      | 优点                 | 缺点                 |
| --------- | -------------------- | -------------------- |
| 数据库    | 实现简单，容易理解   | 对数据库压力大       |
| Redis     | 易于理解             | 自己实现，不支持阻塞 |
| Zookeeper | 支持阻塞             | 程序复杂             |
| Curator   | 提供锁的方法         | 依赖Zookeeper,强一致 |
| Redisson  | 提供锁的方法，可阻塞 |                      |
